# qlizmet

Quizlet-подобное десктопное приложение для заучивания карточек. Курсовая работа.

Поддерживает режимы Quizlet (карточки, Learn с интервальными повторениями, Write,
Test, Match, Gravity), локальное хранение на одного пользователя, карточки с
текстом, картинками и LaTeX-формулами.

## Стек

- Python 3.11+
- [PySide6](https://doc.qt.io/qtforpython/) (Qt) — интерфейс
- SQLite — хранение
- pytest — тесты

Зависимости растут вместе с этапами: рендер формул (matplotlib) появится вместе с
соответствующим этапом, а не тянется в скелет заранее.

## Архитектура

Слоистая, с односторонним правилом зависимостей `ui → app → core`. Ядро (`core/`) —
чистый Python без единого импорта Qt, полностью тестируется без запуска окна.
Хранилище подключается к ядру через интерфейсы (`storage/repository.py`), поэтому
БД остаётся сменной деталью.

```
src/qlizmet/
  core/     модели, режимы, SRS, оценка ответов, статистика (чистый Python)
  storage/  репозитории и их реализации, импорт/экспорт
  app/      сервисы и use-cases
  ui/       Qt: единственный слой, импортирующий PySide6
```

## Запуск

```bash
pip install -e ".[dev]"
python -m qlizmet
```

## Тесты

```bash
pytest
```
